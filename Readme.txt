# excel-appearance
What it Does:
Allows you to hide UI elements on Excel Application


How to Install:
Copy the Appearance_HavishMConsulting.xlam file to the Excel Add-Ins folder.
It should look like this - 
C:\Users\[yourname]\AppData\Roaming\Microsoft\AddIns

Open Excel application > File > Options > Add-ins > Manage > Excel Add-ins > Go
Select Appearance_HavishMConsulting and Press Okay


What you can do:
Hide / Show Gridline, Heading, Formula Bar, Status Bar, Sheet Tabs, Scroll Bars
The Appearance Form can also be Minimized / Maximized
